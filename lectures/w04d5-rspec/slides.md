## RSpec

+ Domain-Specific Language (DSL) written in Ruby for Ruby
+ Meta-gem that includes multiple more specific gems
	+ `rspec-core`
  + `rspec-expectations`
  + `rspec-mocks`
+ Not the only Ruby testing library
	+ Others include `Minitest` and `Cucumber`
